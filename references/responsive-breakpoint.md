## Responsive Breakpoint

| 最小宽度 | Tailwind 类名 | CSS 变量名 | 用途 |
|---------|--------------|-----------|------|
| `375px` | `xs` | `var(--breakpoint-xs)` | 常规手机 |
| `600px` | `sm` | `var(--breakpoint-sm)` | 折叠屏手机、iPad Mini、Pad 竖屏 |
| `960px` | `md` | `var(--breakpoint-md)` | Pad 横屏 |
| `1280px` | `lg` | `var(--breakpoint-lg)` | 13-14 寸笔记本 |
| `1440px` | `xl` | `var(--breakpoint-xl)` | 15-16 寸笔记本、大部分台式机 |
| `1680px` | `2xl` | `var(--breakpoint-2xl)` | 高分辨率大显示器 |
| `1920px` | `3xl` | `var(--breakpoint-3xl)` | 1080p 及以上大显示器、电视屏 |
